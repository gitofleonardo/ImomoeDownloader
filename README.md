# ImomoeDownloader

 Download video from imomoe.jp
 
**Q:咋用啊这东西**

**A:就一个简单到不行的网页封装，到达视频页面后可以点击`check`按钮检查是否有符合格式的视频，或者直接点击`Download`按钮也可以检查，如果通过后直接下载。**

> 注：为了偷懒，就用的系统浏览器来下来，如果以后有时间的话可以考虑自己写一个下载器

> 注：已经实现了下载器，选择保存目录
